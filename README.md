# MyDiscussionForum

Final project for COSC 360 Web Programming.

## Requirements

- [Docker Desktop](https://www.docker.com/get-started)
- [Node LTS](https://nodejs.org/en/)

## Getting Started

- Install dependancies
  ```bash
    cd react-app && npm i
    cd functions && npm i
  ```
- Run dev environment
  ```bash
  ./dev.sh
  ```
